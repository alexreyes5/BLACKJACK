# BLACKJACK
ABOUT: Terminal Blackjack game built in Java using OOP. Special as it is my first project on what I hope to be the start of a skilled programmer/engineer one day.
Project was created right after I finished COMP 110, and wanted to immediately start learning other advanced topics such as ArrayList, which is included in Blackjack. (For this project, Ace is currently set to 11.)

FEATURES
- Custom player name input
- Standard 52-card deck
- Deck shuffling
- Hit and Stand gameplay
- Hidden dealer card
- Win, loss, and tie outcomes

CONCEPTS
- Java
- Object-Oriented Programming (OOP)
- Classes and Objects
- ArrayLists
- Loops and Conditionals
- Scanner/User Input


STRUCTURE

- Main.java – controls game flow
- Card.java – represents a playing card
- Deck.java – creates and manages the deck
- Player.java – stores player hands and calculates totals

LEARNED
- Using multiple classes
- Managing objects and interactions between classes
- Building game logic with loops and conditionals
- Using ArrayLists to store and manage data
- Debugging and testing Java programs

IMPROVEMENTS FOR FUTURE
- Ace value adjustment (1 or 11)
- Replay system
- Betting system
- Additional game stats
